Banco de Dados ⚄


Um Banco de Dados é utilizado para guardar e organizar informações(dados) de um sistema computacional.

É uma disciplina bastante importante no ramo da tecnologia, pois muitos softwares precisam armazenar dados diversos dos sistemas.

Imagine um sistema simples de cadastro de mercadorias em uma loja, nesse caso quem armazena esses cadastrros é justamente um banco de dados.
